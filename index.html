Форум для геймеров

(Ещё дорабатывается, GitHub приватный)

Описание: Данный сайт объединяет две ключевые функции — активный форум и систему проведения турниров для геймеров. Форум предоставляет пользователям возможность задавать вопросы, отвечать на них и оставлять комментарии, создавая живое и дружелюбное сообщество. Турниры позволяют создавать собственные соревнования, а также регистрироваться и участвовать в уже существующих турнирах, что способствует развитию соревновательного духа и взаимодействию игроков. Для удобства и безопасности реализована полноценная система регистрации и авторизации пользователей.

Стек технологий: PHP, JavaScript, HTML/CSS, SQL, MySQL

Трудности при разработке: Оптимизация производительности форума; Обеспечение безопасности и надёжности системы авторизации; Проектирование гибкой системы турниров.

ToDo-List, написанный на React

Описание: Интерактивное приложение на React, позволяющее создавать, редактировать и удалять задачи. Использует Hooks для управления состоянием, localStorage для сохранения данных между сессиями и обеспечивает удобный интерфейс для пользователей. Отличный пример SPA-приложения с современной архитектурой.

Стек технологий: React (Hooks, функциональные компоненты), JavaScript (ES6+), HTML5, CSS3, localStorage API

Трудности при разработке: Управление состоянием задач с учётом добавления, редактирования и удаления без потери производительности; Синхронизация состояния приложения с localStorage; Оптимизация рендеринга компонентов для предотвращения лишних перерисовок.
